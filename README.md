# Dubai Luxury Real Estate — сайт-визитка

Трёхстраничный сайт-визитка брокера элитной недвижимости в Дубае.
Чистый HTML5, CSS3 и vanilla JavaScript — без сборки и зависимостей.

## Структура проекта

```
dubai-broker/
├── index.html       # Главная / Обо мне
├── services.html    # Услуги и направления
├── contacts.html    # Контакты + форма + карта
├── style.css        # Общие стили (адаптивные)
├── script.js        # Скролл-анимации, бургер-меню, счётчики, форма
└── README.md
```

## Как запустить

### Способ 1. Просто открыть в браузере
Двойной клик по `index.html` — сайт откроется в браузере.
Все ссылки между страницами относительные и будут работать.

### Способ 2. Локальный сервер (рекомендуется)
Чтобы корректно работали Google Fonts, Unsplash и iframe Google Maps, лучше запустить локальный сервер:

```bash
# Python 3
cd dubai-broker
python3 -m http.server 8000
# открыть http://localhost:8000

# или Node.js
npx serve .
```

## Особенности

- Премиум-эстетика: тёплый кремовый фон, приглушённое золото, Playfair Display + Manrope.
- Адаптивная вёрстка: ломается под планшеты (≤1024px) и мобильные (≤768px, ≤480px).
- Шапка становится белой при скролле, на мобильных — бургер-меню.
- Плавные `IntersectionObserver`-анимации появления секций.
- Анимированные счётчики на главной.
- Форма на странице контактов формирует ссылку на WhatsApp с предзаполненным текстом — без бэкенда.
- Встроенная карта Google Maps указывает на Golden Mile 8, Palm Jumeirah.

## Кастомизация

- Цвета меняются в CSS-переменных в `:root` файла `style.css`.
- Изображения подгружаются с Unsplash по прямым URL — можно заменить на свои.
- Тексты, статистика и адрес правятся напрямую в HTML.
